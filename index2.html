Learning Reflections on Problems in Nature

Iteration Problems

Execution of the same steps until the condition is satisfied or completed:

  • Example: Heartbeat - The regular pumping of blood is an iterative process.

Recursion Problems

Problems solved by breaking them into smaller parts:

  • Example: DNA Replication - DNA strands replicate themselves in a recursive manner.

Backtracking Problems

Exploring all possibilities and backtracking when a solution path proves invalid:

  • Example: Hunting by Predators - Predators adapt strategies when one fails to catch prey.

Parallelism in Nature

Involves multiple tasks executing simultaneously:

  • Example: Photosynthesis.

Space and Time Efficiencies

  • Space Efficiency: The amount of memory an algorithm uses during execution. Example: Quick Sort.
  • Time Efficiency: The time an algorithm takes based on input size:
    • O(1): Quick, like looking up a name in a small list.
    • O(n): Proportional to input size, like scanning through a book.
    • O(2‚Åø): Very slow, like testing every move in chess.

Key Techniques

  • Randomized Algorithms: Use random numbers to influence behavior.
  • Recursion: Functions call themselves to solve smaller problems.
  • Backtracking: Explore all possible solutions, discarding invalid paths.
  • Dynamic Programming: Avoid redundant computations by storing results of sub-problems.

Hierarchical Data and Trees

  • AVL Tree: Balanced for better performance.
  • Red-Black Tree: Faster in some cases than AVL.
  • Heap: Used for priority tasks like job scheduling.
  • Tries: Ideal for autocomplete or dictionary lookups.

Array Query Algorithms

  • Efficiently retrieve or update data in arrays. Applications: Database queries, real-time data, gaming.
  • Key Techniques: Prefix Sum, Sliding Window.

Trees vs. Graphs

  • Tree: Single root, no cycles, directed edges.
  • Graph: No root, can have cycles, edges may be directed or undirected.

Sorting and Searching

  • Sorting: Bubble Sort, Selection Sort, Insertion Sort, Quick Sort.
  • Searching: Linear Search, Binary Search, Exponential Search.
